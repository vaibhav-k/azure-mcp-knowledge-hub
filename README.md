# Azure MCP Knowledge Hub

A Model Context Protocol (MCP) based knowledge platform built with Azure services.

## Current Status

🚧 Under active development

## Architecture

The platform will contain:

- MCP Client
- Document MCP Server
- Employee MCP Server
- Azure Blob Storage
- Azure SQL Database
- Azure Infrastructure as Code

## Tech Stack

- Python
- FastMCP
- Azure Blob Storage
- Azure SQL
- Azure Bicep

## Document MCP Server

The first MCP server provides access to documents stored in Azure Blob Storage.

### Available Tools

| Tool | Description |
|------|-------------|
| list_documents | Lists files from an Azure Blob Storage container |

### Local Run

```bash
python document_server/app.py
```
