# Azure MCP Knowledge Hub

A Model Context Protocol (MCP) based knowledge platform built with Azure services.

🚧 Under active development

## Architecture

The platform will contain:

- MCP Client
- Document MCP Server
- Employee MCP Server
- Azure Blob Storage
- Azure SQL Database
- Azure Infrastructure as Code

- FastMCP
